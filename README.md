# Sales-Data-Analysis-Dashboard
### 1.	Project Title / Headline
An interactive Power BI dashboard designed to analyze sales performance, customer behavior, product trends, and regional profitability across a supermarket business. The dashboard provides actionable insights into revenue generation, customer purchasing patterns, shipping preferences, and business growth trends to support data-driven decision-making.

### 2.	Short Description / Purpose
The Super Market Sales Dashboard helps business stakeholders monitor key sales metrics, identify high-performing products and customer segments, and evaluate regional sales performance. Through interactive visualizations and KPIs, users can quickly uncover trends and opportunities for business growth.

This dashboard is ideal for:

Business Analysts
Sales Managers
Marketing Teams
Supply Chain Managers
Data Analysts

###3.	Tech Stack
The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Primary data visualization and dashboard development platform.
🔄 Power Query – Used for data cleaning, transformation, and preparation.
🧠 DAX (Data Analysis Expressions) – Used for calculated measures, KPIs, and business logic.
📈 Data Modeling – Relationships established between sales, customers, products, and regional datasets.
🗺️ Power BI Maps – Used for geographical sales analysis across states.
📁 File Format – .pbix for development and .png for dashboard previews.

### 4.	Data Source
Dataset: Super Store Sales Dataset

The dataset contains transactional sales data including:

Order Information
Product Categories and Sub-Categories
Customer Segments
Payment Modes
Shipping Modes
Regional and State-Level Sales
Profit Metrics
Delivery Performance

The data was cleaned and transformed using Power Query before creating relationships and DAX measures in Power BI.

### 5.	Features / Highlights
Business Problem

Retail businesses generate large volumes of sales data daily, making it difficult to identify:

Which products drive the highest revenue?
Which customer segments contribute most to sales?
Which regions perform best?
What payment methods are preferred by customers?
How sales and profits change over time?
Which shipping methods are most frequently used?

Without a centralized analytical solution, extracting meaningful insights becomes time-consuming and inefficient.
